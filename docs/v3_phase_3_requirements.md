作成日: 2026-06-07  
更新日: 2026-06-08  
対象: `v3/`  
位置づけ: V3.1 main昇格前の算数拡張要求仕様

## 1. 全体像と現在地

```text
V3 Phase 1: 完了
V3 Phase 2: ES Modules化完了
V3 Phase 3 / V3.1: 算数レベル5・6追加 + main昇格判断 ← 本ドキュメント
```

当初の Phase 3 では、割り算と国旗クイズを同一フェーズで扱う案としていた。

その後、リリースリスクを分離するため、V3.1 / Phase 3 では算数レベル5・6追加までを対象とし、国旗クイズは V3.2 / Phase 4 に分離する方針へ変更する。

## 2. Phase 3 / V3.1 の目的

V3.1 / Phase 3では、PokeMathの既存算数クイズを維持しつつ、割り算クイズを追加する。

主目的は以下。

```text
1. 既存の算数クイズ仕様を維持する
2. 算数クイズにレベル5・6を追加する
3. V3を算数拡張版として安定化する
4. 実機確認後、V3の main昇格判断へ進める状態にする
```

## 9. レベル5: 九九のわり算

### 9.1 目的

九九の逆算として、余りのない割り算を出題する。

### 9.2 問題例

```text
12 ÷ 3 = ?
24 ÷ 6 = ?
56 ÷ 7 = ?
```

### 9.3 生成ルール

```text
答え: 1〜9
割る数: 1〜9
割られる数: 答え × 割る数
```

### 9.4 要求

```text
1. 必ず割り切れること
2. 答えは1〜9であること
3. 4択で回答できること
4. 既存の報酬仕様に乗ること
5. 不正解時に正解が表示されること
```

## 10. レベル6: 2けた ÷ 1けた の余りなし割り算

### 10.1 目的

九九のわり算の次段階として、2けたの数を1けたで割る問題を出題する。

### 10.2 問題例

```text
36 ÷ 3 = ?
48 ÷ 4 = ?
84 ÷ 7 = ?
96 ÷ 8 = ?
```

### 10.3 生成ルール

初期版では、難易度を抑える。

```text
割る数: 2〜9
答え: 2〜12
割られる数: 割る数 × 答え
条件: 割られる数が2けたであること
```

### 10.4 要求

```text
1. 必ず割り切れること
2. 割られる数は2けたであること
3. 割る数は1けたであること
4. 初期版では答えは2〜12程度に抑えること
5. 4択で回答できること
6. 既存の報酬仕様に乗ること
7. 不正解時に正解が表示されること
```
