# ポケモンさんすうアプリ v1.5 仕様定義書

> 注記: この文書は V1.5 の仕様定義書です。現在の root `index.html` は V2 Main に昇格済みです。V1.5実装は `archive/index_v1_5.html` に退避済みであり、本書はV1.5参照・アーカイブ用として維持します。

## 1. アプリケーション概要

本アプリケーションは、子どもが楽しく算数の学習を継続できるよう、クイズの成果に応じて「ポケットモンスター（初代151匹）」を収集・図鑑登録できるモバイルファーストのWebアプリケーション（SPA）である。

## 2. 機能要件

### 2.1 画面構成

1. タイトル画面
2. プレイヤー選択画面
3. メインメニュー画面
4. クイズ画面
5. 結果発表画面
6. ポケモン図鑑画面

### 2.2 ユーザー管理

- ユーザーは「りょうま」「さら」「ゲスト」。
- Firestore `users/{username}` に取得済みポケモンを保存する。
- 保存フィールドは主に `owned`。

### 2.3 クイズ仕様

- 1セッション5問。
- 4択クイズ。
- レベル1〜4を持つ。
- 4問正解で1匹、5問正解で2匹を取得する。

### 2.4 図鑑仕様

- 初代151匹を対象とする。
- 未取得ポケモンは `?` / `???` として隠す。
- 取得済みポケモンは詳細モーダルで表示する。

## 3. V2との関係

| 項目 | V1.5 | V2 Main |
| --- | --- | --- |
| 本体ファイル | `archive/index_v1_5.html` | `index.html` |
| 対象ポケモン | 第1世代151匹 | 第1〜第9世代1025匹 |
| ユーザーデータ | `users/{username}.owned` | `users_v2/{userId}` |
| マスターデータ | 起動時にPokeAPI取得 | `masters/gen_{1..9}` を読み取り |
| 世代進行 | 未実装フック | 実装済み |
| 図鑑 | 第1世代固定 | クリア済み世代 + 現在世代 |
